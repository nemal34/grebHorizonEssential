F4SE Mods

Buffout 4

Instructions 1. Download and install with your mod manager the first file (Buffout 4) from the Buffout 4 page
2. Download manually and extract to your root folder (where Fallout4.exe is) the second file (TBB redistributables) from the Buffout 4 page
3. Download and install with your mod manager Address Library for F4SE Plugins
4. Download manually and extract to your root folder (where Fallout4.exe is) xSE PluginPreloader F4
5. Open xSE PluginPreloader.xml with a text editor (I recommend Notepad++) and on line 14 change OnProcessAttach to OnThreadAttach
6. Download and run vc_redist.x64.exe from the Microsoft Visual C++ Redistributables for Visual Studio 2019 page
7. If you're using the Blur Removal plugin from Horizon, download the fixed plugin pinned in the Discord #mod-compatibility channel
